# Coworking Hub Manager

Coworking Hub Manager is a web application for managing and booking shared workspaces, offices, and meeting rooms.

## Features
- User roles:
  - Member
  - Space Manager
  - System Administrator

## Core Functionality
- Authentication and registration with approval system
- Password reset via temporary link (valid for 30 minutes)
- Profile management with image upload
- Search and filtering of coworking spaces

## Booking System
- Reservation of:
  - Open workspace desks
  - Private offices
  - Conference rooms
- Interactive calendar view
- Prevention of double bookings

## Member Features
- Booking history and management
- Like/dislike system and comments 
- Personalized interaction with spaces

## Manager Features
- Workspace creation and configuration
- JSON-based bulk import of spaces
- Reservation tracking and attendance confirmation
- Drag-and-drop calendar management
- Monthly PDF reports 

## Admin Features
- User approval and management
- Workspace approval system
- Analytics dashboards (charts for popularity and revenue)

## Tech Stack
- Frontend: Angular 20
- Backend: Node.js + Express
- Database:MongoDB

## Project Components
- Full-stack implementation
- Database setup (predefined and populated)
- UI/UX design
- System architecture design
